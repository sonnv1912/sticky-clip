# Sticky Clip Document

Web document site for Sticky Clip. This app lives inside the Sticky Clip workspace and shares packages from `../../packages`.

## Development

From the workspace root:

```bash
yarn dev:document
```

Or from this directory:

```bash
yarn dev
```

The app runs on `http://localhost:3333`.

## Build

From the workspace root:

```bash
yarn build:document
```

The production output is written to `dist/`.
